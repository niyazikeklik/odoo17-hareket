PREVIEW ALL TYPES OF ATTACHMENT

Preview All Types of Attachment saves effort and storage of users as it allows them to preview the document before downloading it. This app can be used in all Odoo modules to display attachments including MS Office, PDF, image, video file format.

The Preview Attachments module is a combination of two modules Preview MS Office Attachments and Preview Many2many Attachments of Win DX. Please do not install the Preview Attachments module and one of the two modules above at the same time to avoid system errors.

FEATURES

  • Supports MULTIPLE file format such as MSOffice, image, PDF and video
  • Preview files from ANYWHERE inside the Odoo module
  • User can preview a document WITHOUT DOWNLOADING
HOW IT WORKS

1. Works with custom many2many fields

  • If you’re adding the Many2Many Field to a new code or already working code, then add widget="many2many_preview_attachment" in xml view wherever you wish to add the field.
  • A button like ( ) will be visible beside all binary fields like shown in the image.
  • Allow access to preview file formats include docx, xlsx, pptx, pdf, jpeg, gif, png and mp4 as attachments for the normal user.

2. Basic display

  • Right after installing this module, a normal user can preview what they attached in the “Files” field.
